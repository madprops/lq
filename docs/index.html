List directories

Flags:

  --dirs
  Just show directories
  --files
  Just show files
  --absolute (or -a)
  Use absolute paths
  --no-colors
  Don't color paths
  --dev
  Used for development
  --prefix (or -p)
  Use prefixes like '[F]'
  --list (or -l)
  Show in a vertical list
  --count (or -c)
  Count items inside directories
  --no-titles
  Don't show titles like 'Files'
  --reverse (or -r)
  Put files above directories
  --no-spacing (or -x)
  Make it less comfy
  --fluid
  Don't put linebreaks between sections
  --mix
  Mix and sort everything
  --abc
  Categorize with letters
  --size (or -z)
  Show the size of files
  --sizesort
  Sort by file size
  --datesort (or -d)
  Sort by file modification date
  --salad (or -s)
  Preset to mix all
  --blender (or -b)
  Preset to really mix all

Values:

  --filter (or -f)
  Filter the list.
  Start with re: to use regex.
  For instance --filter=re:\\d+

Arguments:

  path
  Path to a directory
