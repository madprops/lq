List directories

Flags:

  --dirs (or -1)
  Just show directories

  --files (or -2)
  Just show files

  --execs (or -3)
  Just show executables

  --absolute (or -a)
  Use absolute paths

  --prefix (or -p)
  Use prefixes like '[F]'

  --list (or -l)
  Show in a vertical list

  --count (or -c)
  Count items inside directories

  --no-titles (or -x)
  Don't show titles like 'Files'

  --reverse (or -r)
  Put files above directories

  --fluid (or -u)
  Don't put linebreaks between sections

  --mix (or -m)
  Mix and sort everything

  --abc (or -@)
  Categorize by letters

  --size (or -z)
  Show the size of files

  --dirsize (or -Z)
  Show the size directories

  --date (or -k)
  Show the last modification date on files

  --dirdate (or -K)
  Show the last modification date on directories

  --sizesort (or -i)
  Sort files by size

  --dirsizesort (or -I)
  Sort directories by size

  --datesort (or -d)
  Sort files by modification date

  --dirdatesort (or -D)
  Sort directories by modification date

  --header (or -h)
  Show a header with some information

  --permissions (or -P)
  Show posix permissions

  --tree (or -t)
  Show directories in a tree structure

  --ignore-config (or -!)
  Don't read the config file

  --ignore-dots (or -#)
  Don't show dot dirs/files

  --reverse-sort (or -R)
  Reverse sorting

  --snippets (or -S)
  Show text file snippets

  --mix-files (or -M)
  Mix files and executables

  --salad (or -s)
  Preset to mix all

  --blender (or -b)
  Preset to really mix all

  --info (or -?)
  Preset to show some information

  --dev
  Used for development

Values:

  --filter (or -f)
  Filter the list.
  Start with re: to use regex.
  For instance --filter=re:\\d+

  --exclude (or -e)
  Directories to exclude

  --max-width (or -w)
  Maximum horizontal size

  --output (or -o)
  Path to a file to save the output

  --snippets-length (or -n)
  Max length of snippets

Arguments:

  path
  Path to a directory

Notes:

  A config file should be in ~/.config/lq

  Git Repo: https://github.com/madprops/lq
